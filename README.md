# static_vector

stack-allocated vector that mimics std::vector interface

### Tested for compatibility:

- standards:
    - c++17
    - c++20
    - c++23
- compilers:
    - icc-2024.1
    - gcc-10
    - gcc-11
    - gcc-12
    - gcc-13
    - gcc-14
    - clang-15 (partial, benchmark fail to compile)
    - clang-16
    - clang-17 (partial, benchmark fail to compile)
    - clang-18
